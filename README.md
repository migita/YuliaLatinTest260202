# Latin → English Quiz (GitHub Pages)

Static, no backend. Progress is saved in the browser via `localStorage`.

## Files

- `index.html` – app UI
- `style.css` – styling
- `data.js` – vocabulary + grammar items (from your uploaded sheets)
- `script.js` – quiz logic

## Publish on GitHub Pages (quick)

1. Create a GitHub repository (public or private).
2. Upload these files to the repository root.
3. In GitHub: **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: **main** (or master) / **root**
4. Open the Pages URL GitHub gives you.

## Notes

- Works on mobile.
- “Typing” mode is used for vocabulary; grammar questions are shown as multiple-choice even if typing mode is selected.
- Export/Reset buttons are on the start screen.
